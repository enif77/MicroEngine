# Micro Engine

A miniature game engine written in C#, using OpenTk.


## Features

- [x] Scene graph.
- [x] Lights.
- [x] Materials.
- [x] Textures.
- [x] Meshes.
- [x] Cameras.
- [x] Skybox. 
- [ ] Shadows.
- [ ] Physics.
- [ ] Audio.
- [ ] Scripting.
