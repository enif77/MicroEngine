# Micro Engine

A miniature game engine written in C#, using OpenTk.

![Main build workflow status](https://github.com/enif77/MicroEngine/actions/workflows/dotnet.yml/badge.svg)

## Features

- [x] Scene graph.
- [x] Lights.
- [x] Materials.
- [x] Textures.
- [x] Meshes.
- [x] Cameras.
- [x] Skybox. 
- [ ] Shadows.
- [ ] Physics.
- [ ] Audio.
- [ ] Scripting.
